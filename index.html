Laramce

A Laravel bundle to provide the TinyMCE editor.

Download .zip Download .tar.gz View on GitHub

LaraMCE Bundle, by Charl Gottschalk

LaraMCE allows you to generate rich text boxes based on TinyMCE for the excellent Laravel PHP framework

Update: Now supports all TinyMCE configuration settings

1. Install using Artisan CLI:
php artisan bundle:install laramce
2. Add the following line to application/bundles.php file:
return array('laramce' => array('auto' => true),);
3. Add the following to the application.php config file in the 'aliases' array:
'RTE'                 => 'Laramce\\RTE',
4. Publish the bundle assets to your public folder:
php artisan bundle:publish
5. Add the following to your template view file to include the TinyMCE Javascript:
Asset::container('laramce')->scripts();
To create a rich text box:

For the mode setting, use 'simple' to create a simple editor, 'full' to create an editor with all options enabled and 'custom' to create an editor with your own settings

RTE::rich_text_box('editorName', 'editorValue',array(
             'selector'=>'editorSelector',
             'mode'=>'full',
             'att'=>'array('id'=>'editorId'),
             'setup'=>array(
                'skin'=>'o2k7',
                'skin_variant'=>'black')
            ))
To create initialization script:

Use this method if you want to create multiple editors on a single page or generate the script where you want.
When using this function, you can create normal textareas using Laravel's form builder, just remember to add the same selector id for the script as a class attribute to the textarea.

RTE::initialize_script(array('selector'=>'editorSelector',
                 'mode'=>'full',
                 'setup'=>array(
                     'skin'=>'o2k7',
                     'skin_variant'=>'black')           
                 ))
LaraMCE settings:
'selector'  =>   'a unique name that TinyMCE will use to select the textarea',

Required

'mode'      =>   'full/simple/custom',

Refer to TinyMCE configuration documentation for correct setting names and values - http://www.tinymce.com/wiki.php/Configuration 

'setup'     =>   array( 'TinyMCE Setting'=>'Value', 'Another TinyMCE Setting'=>'Value')

Current TinyMCE version is 3.5.6
